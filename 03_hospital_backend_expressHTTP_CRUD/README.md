# Hospital Backend Express API

A simple backend API for hospital management using Express.js.

## Features

- Get kidney health information for patients
- RESTful API endpoints for CRUD operations

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the server:
   ```bash
   npm start
   ```

2. The server will run on `http://localhost:3000`

## API Endpoints

- `GET /` - Get kidney health information
- `POST /` - Add new kidney data (to be implemented)
- `PUT /` - Update kidney health status (to be implemented) 
- `DELETE /` - Remove kidney data (to be implemented)

## Technologies Used

- Node.js
- Express.js
